1. DESIGN PHILOSOPHY
----------------------------------------------------------------

StreakSky's visual design is built on four principles:

  DARK & POWERFUL
  The app lives in dark mode by default. Dark backgrounds make
  neon accent colors pop with energy. It feels premium, focused,
  and distraction-free — like a professional tool, not a toy.

  NEON ENERGY
  A single electric accent color (neon green) drives all
  interactive elements, progress indicators, active states,
  and celebration moments. One accent. Used with intention.

  DATA AS ART
  Charts, heatmaps, and graphs are not utility elements.
  They are the hero visuals of every screen. They should
  feel like looking at something beautiful, not a spreadsheet.

  ZERO CLUTTER
  Every screen has one primary action. Generous white space
  (dark space). Typography does the heavy lifting. Icons
  are minimal and consistent.

3. TYPOGRAPHY
----------------------------------------------------------------

FONT FAMILY
  Primary Font        : Inter (or SF Pro on iOS)
  Display / Hero      : Inter Black (weight 900)
  Headings            : Inter Bold (weight 700)
  Body                : Inter Regular (weight 400)
  Labels / Metadata   : Inter Medium (weight 500)

TYPE SCALE
  Hero Number         : 64px  Bold    (streak count, big stats)
  Display             : 32px  Black   (screen titles, year)
  H1                  : 24px  Bold    (section headers)
  H2                  : 20px  Bold    (card titles)
  H3                  : 17px  SemiBold(sub-section labels)
  Body Large          : 16px  Regular (main body text)
  Body                : 14px  Regular (descriptions)
  Caption             : 12px  Medium  (metadata, timestamps)
  Micro               : 10px  Medium  (heatmap labels, badges)

SPECIAL TEXT RULES
  - All numbers (streaks, stats, counts) use tabular figures
    so digits don't shift width as numbers change
  - Quote text uses 18px Regular with 1.6 line height
    for readability and elegance
  - Habit names on cards: 15px SemiBold, never truncated
    below 2 lines
  - Uppercase used ONLY for section labels and badge names
  - Letter spacing: +0.5px on all uppercase labels

8. ANIMATIONS & MICRO-INTERACTIONS
----------------------------------------------------------------

  HABIT COMPLETION
  - Checkbox tap: scale bounce (1.0 → 1.2 → 1.0, 200ms)
  - Neon glow pulse radiates outward from card (300ms)
  - Haptic: medium impact
  - If all habits done: full screen neon flash + ☀️ animation

  STREAK MILESTONE
  - Badge slides up from bottom with spring animation
  - Background dims to 60% black
  - Badge glows and pulses 3 times
  - Quote fades in below badge
  - Haptic: success pattern

  HEATMAP LOAD
  - Cells fade in left to right, row by row (600ms total)
  - Each cell: opacity 0 → 1 with slight scale

  WEATHER CHANGE
  - Weather icon on home screen: crossfade with scale
  - Background gradient shifts to match weather color

  GRAPH DRAW
  - Line graphs draw from left to right (400ms)
  - Bar charts grow from bottom up (300ms)
  - Data points pop in with spring scale

  YEAR REVIEW REVEAL
  - Cinematic: dark screen → year number fades in
  - Heatmap cells light up progressively
  - Stats count up from 0 to final number
  - AI narrative types out character by character

  PAGE TRANSITIONS
  - Standard: slide left/right (iOS native)
  - Modal: slide up from bottom
  - Year review: fade in (special moment)

  LOADING STATES
  - Skeleton screens (dark shimmer, no spinners)
  - Shimmer color: #2E2E2E → #3A3A3A animated

----------------------------------------------------------------
10. ACCESSIBILITY
----------------------------------------------------------------

  - Minimum contrast ratio: 4.5:1 for all body text
  - Neon green (#B3FF00) on black (#0D0D0D): 13.2:1 ratio
    (AAA compliant — well above minimum)
  - All interactive elements minimum 44×44px tap target
  - Dynamic type support (iOS) — text scales with system
  - Reduced motion: all animations respect
    prefers-reduced-motion setting
  - Screen reader labels on all icons and charts
  - Color-blind mode: replace green/red with
    shape differentiation on heatmap cells

12. DESIGN INSPIRATION REFERENCE SUMMARY
----------------------------------------------------------------

The Pinterest reference image shows a dark fitness app with:
  - Pure near-black backgrounds (#0D0D0D family)
  - Single electric green accent (#B3FF00 equivalent)
  - Clean card-based layout with generous spacing
  - Mini bar charts and waveform charts inside stat cards
  - Profile page with avatar ring, menu rows, chevrons
  - Edit profile with labeled form fields, save + delete
  - Login screen with centered layout, subtle inputs
  - Activity calendar strip (7-day horizontal)
  - Statistics screen with 2×2 metric card grid
  - 78+ total screens confirming comprehensive coverage

StreakSky adopts ALL of these visual decisions and adapts
the content from fitness/workout tracking to habit tracking,
goal cascading, streak management, and AI-powered coaching.
The design language is identical. The product is deeper.
